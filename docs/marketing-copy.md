### Product descriptions

**One-liner (10 words):**
> Desktop automation tool — mouse, keyboard, apps, browser, scripts.

**Elevator pitch (30 words):**
> Desktop Automator runs sequences of actions on your Windows PC — opening apps, clicking, typing, browsing, running scripts — all configurable, all unattended, all without writing code.

**Short paragraph (60 words):**
> Stop doing the same things over and over. Desktop Automator lets you build custom workflows that run automatically: launch applications, navigate websites, move the mouse naturally, type with human-like speed, execute PowerShell scripts, and capture screenshots. Configure once, run forever. Lightweight (5 MB), private (100% local), and fast (built with Rust).

**Full description (150 words):**
> Desktop Automator is a professional desktop automation tool for Windows that executes complex sequences of actions without requiring programming knowledge. Create workflows by combining 17+ activity types: open applications, browse URLs, move the mouse with natural bezier curves, type text at variable speed, click at exact screen coordinates, run PowerShell scripts, manage files, and more. Use dynamic variables like {{date}}, {{time}}, and {{prev_result}} to chain activities together intelligently. Save different profiles for different routines. The app runs silently in the system tray, uses only 15 MB of RAM, and keeps all data 100% local on your machine — nothing is ever transmitted. Built with Tauri 2.0 and Rust for maximum performance. Supports English and Spanish. Perfect for QA testing, automated demos, kiosk displays, session management, and any repetitive desktop workflow you want to eliminate.
