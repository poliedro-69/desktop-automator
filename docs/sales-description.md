## The Solution

**Activity Simulator** runs quietly in the background and performs realistic human-like activity on your Windows PC:

- 🖥️ **Launches applications** on a schedule
- 📊 **Opens spreadsheets and documents** with their default programs
- 🌐 **Browses URLs** — including your corporate intranet
- 🖱️ **Moves the mouse** along natural, curved paths with scrolling and clicks
- ⌨️ **Types text** at human speed, complete with occasional typos and corrections
- 📸 **Takes screenshots** with timestamps for your own records
- ⚡ **Runs PowerShell scripts** silently in the background
- ⏸️ **Takes realistic breaks** so the pattern never looks robotic

Everything is configurable. Set the delay between actions, how many cycles to run, and toggle individual activities on or off. Save your setup as a profile and load it in one click.

## Why It Looks Real

Most simulators move the mouse in straight lines at fixed intervals. Anyone watching can spot it immediately.

Activity Simulator uses **eased bezier-style movement curves**, random micro-pauses between actions, variable typing speed, and occasional self-correction — the same subtle imperfections that make human input recognizable.

No two sessions look the same.

## FAQ

**Is this legal?**
Activity Simulator is a desktop automation tool. Like any tool, its legality depends on how you use it. Using it on your own hardware, for demos, testing, or personal productivity is perfectly fine. Using it to violate employment contracts or deceive employers is your responsibility, not ours. Check your company's policies.

**Will antivirus flag it?**
The installer is code-signed. Modern antivirus tools look for malicious behavior — Activity Simulator doesn't inject into other processes, doesn't access the network (except for license validation), and doesn't touch system files.

**Does it work with dual monitors?**
Yes. Mouse activity covers all connected screens. Screenshots capture each display separately.

**Can I automate a specific workflow?**
Yes. Build a custom sequence of activities, set the timing, save it as a profile. Load different profiles for different use cases.

**Does it run on startup automatically?**
Not by default, but you can add it to Windows startup manually or via the included PowerShell activity type.
