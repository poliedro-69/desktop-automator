Desktop automation tool: schedule mouse, keyboard, apps, and browser actions on Windows.

### Long description
```
Desktop Automator is a powerful desktop automation tool for Windows that lets you create and run sequences of automated actions — all without writing code.

Build custom workflows by combining activities like opening applications, browsing URLs, moving the mouse with natural curves, typing text at human speed, clicking at specific screen coordinates, running PowerShell scripts, and more.

KEY FEATURES:

• 17+ automation activities across 6 categories
• Natural mouse movement with bezier curves, scrolling, and random clicks
• Human-like typing with variable speed and self-correction
• Keyboard shortcut execution (Ctrl+S, Alt+Tab, any combination)
• Browser automation — open URLs, rotate between tabs automatically
• PowerShell script execution in the background
• Screenshot capture with timestamps
• File operations — open, copy, organize files automatically
• Dynamic variables and templates ({{date}}, {{time}}, {{counter}}, {{prev_result}})
• Result chaining — pass output from one activity to the next
• Custom variables — define and reuse values across your workflow
• Configurable timing with random delays for natural pacing
• Infinite loop or fixed cycle count
• Save and load profiles (.json) for different workflows
• 8 built-in sample profiles to get started immediately
• Real-time activity log with color-coded status
• System tray mode — runs silently in the background
• Light and dark theme with system detection
• Resizable panels with persistent layout
• Live cursor coordinate display
• Bilingual interface (English / Spanish)
• Lightweight: ~5 MB installed, ~15 MB RAM usage

IDEAL FOR:

• QA and UI testing workflows
• Automated product demos and kiosk displays
• Keeping remote desktop sessions active
• Repetitive task automation
• File backup routines with timestamps
• Scheduled browser navigation
• Workflow prototyping without coding

Built with Tauri 2.0, Rust, and React for maximum performance and minimal resource usage. Runs 100% locally — no data leaves your machine.
```
