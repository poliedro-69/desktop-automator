# Privacy Policy — Desktop Automator

**Last updated:** July 8, 2026

## 1. Introduction

Desktop Automator ("the Software", "we", "us") is a desktop automation tool for Windows developed by [Your Company Name]. This Privacy Policy explains what data the Software collects, how it is used, and your rights regarding your information.

We are committed to protecting your privacy. The Software is designed to operate primarily offline, with minimal data transmission.

---

## 2. Data We Collect

### 2.1 Data stored locally on your device

The following data is stored **exclusively on your computer** and never transmitted to our servers:

| Data | Purpose | Storage location |
|---|---|---|
| Activity configurations | Your automation workflows | Saved profile files (.json) |
| Application preferences | Theme, language, panel width | Browser localStorage |
| Recent profiles list | Quick access to recent files | Browser localStorage |
| Screenshots (if enabled) | Taken by the screenshot activity | Folder you specify |

### 2.2 Data transmitted to our servers

The Software may transmit the following data **only** for license validation purposes:

| Data | Purpose | When |
|---|---|---|
| Machine ID (hardware hash) | License verification | At activation and periodic checks |
| License key | Validate purchase | At activation and periodic checks |
| App version | Compatibility checks | With license validation |
| IP address | Standard server communication | With any network request |

**We do NOT collect:**
- Your automation activity content (what you automate)
- File names, paths, or contents you open
- URLs you browse
- Text you type
- Screenshots you take
- Mouse coordinates or movement patterns
- PowerShell script content or output
- Keyboard shortcuts you configure
- Any data from other applications on your system

- Analytics beyond license verification

## 4. Data Sharing

We do **not** sell, rent, or share your personal data with third parties, except:

- **Payment processors** (Stripe, Paddle, LemonSqueezy, or equivalent): process your purchase transaction. They have their own privacy policies.
- **Law enforcement**: only if legally required by a valid court order in our jurisdiction.

---

## 5. Data Security

- All network communication uses **HTTPS/TLS 1.3** encryption.
- License keys are stored encrypted in the Windows Registry.
- Machine IDs are one-way hashed (SHA-256) — the original hardware serials cannot be reconstructed.
- Our validation server does not store IP addresses beyond the duration of the request.

| Data | Retention period |

Desktop Automator is designed to function **fully offline** after initial activation. The Software:

- Does not stream data to cloud services
- Only contacts our server for periodic license re-validation (approximately once every 7 days when online)

If the Software cannot reach our server, it continues to function normally during a grace period.

# Política de Privacidad — Desktop Automator

**Última actualización:** 8 de julio de 2026

## 1. Introducción

Desktop Automator ("el Software", "nosotros") es una herramienta de automatización de escritorio para Windows desarrollada por [Nombre de tu Empresa]. Esta Política de Privacidad explica qué datos recopila el Software, cómo se utilizan y tus derechos sobre tu información.

Nos comprometemos a proteger tu privacidad. El Software está diseñado para operar principalmente sin conexión, con transmisión mínima de datos.

---

## 2. Datos que recopilamos

### 2.1 Datos almacenados localmente en tu dispositivo

Los siguientes datos se almacenan **exclusivamente en tu ordenador** y nunca se transmiten a nuestros servidores:

| Dato | Propósito | Ubicación |
|---|---|---|
| Configuraciones de actividades | Tus flujos de automatización | Archivos de perfil (.json) |
| Preferencias de la app | Tema, idioma, ancho de panel | localStorage del navegador |
| Lista de perfiles recientes | Acceso rápido a archivos recientes | localStorage |
| Capturas de pantalla (si se activan) | Tomadas por la actividad de captura | Carpeta que especifiques |

### 2.2 Datos transmitidos a nuestros servidores

El Software puede transmitir los siguientes datos **únicamente** para verificación de licencia:

| Dato | Propósito | Cuándo |
|---|---|---|
| Machine ID (hash de hardware) | Verificación de licencia | Al activar y comprobaciones periódicas |
| Clave de licencia | Validar la compra | Al activar y comprobaciones periódicas |
| Versión de la app | Verificación de compatibilidad | Con la validación de licencia |
| Dirección IP | Comunicación estándar de red | Con cualquier petición de red |

**NO recopilamos:**
- El contenido de tu automatización (qué automatizas)
- Nombres de archivos, rutas o contenidos que abres
- URLs que navegas
- Texto que escribes
- Capturas de pantalla que tomas
- Coordenadas o patrones de movimiento del mouse
- Contenido o salida de scripts PowerShell
- Atajos de teclado que configuras
- Ningún dato de otras aplicaciones de tu sistema

---

## 3. Cómo usamos tus datos

| Propósito | Base legal |
|---|---|
| Validación de licencia | Ejecución del contrato (tu acuerdo de compra) |
| Prevenir redistribución no autorizada | Interés legítimo |
| Proporcionar actualizaciones | Ejecución del contrato |
| Responder a solicitudes de soporte | Ejecución del contrato |

**NO** usamos tus datos para:
- Publicidad o perfilado de marketing
- Venta a terceros
- Seguimiento de comportamiento
- Analítica más allá de la verificación de licencia

---

## 4. Compartir datos

**NO** vendemos, alquilamos ni compartimos tus datos personales con terceros, excepto:

- **Procesadores de pago** (Stripe, Paddle, LemonSqueezy o equivalente): procesan tu transacción de compra. Tienen sus propias políticas de privacidad.
- **Autoridades legales**: solo si es legalmente requerido por una orden judicial válida en nuestra jurisdicción.

## 5. Seguridad de los datos

- Toda comunicación de red usa cifrado **HTTPS/TLS 1.3**.
- Las claves de licencia se almacenan cifradas en el Registro de Windows.
- Los Machine IDs se hashean unidireccionalmente (SHA-256) — los seriales originales del hardware no pueden reconstruirse.
- Nuestro servidor de validación no almacena direcciones IP más allá de la duración de la petición.
